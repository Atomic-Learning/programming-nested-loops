What a Nested Loop Is

A nested loop is a loop inside another loop. For each iteration of the outer loop, the inner loop runs through all of its iterations.

General Pattern

FOR each outer_value IN outer_collection:
    FOR each inner_value IN inner_collection:
        process(outer_value, inner_value)

How Execution Scales

Nested loops can increase the total number of iterations quickly. If the outer loop runs $m$ times and the inner loop runs $n$ times, the inner body runs $m \times n$ times.

Example

To print all pairs of items from two lists:

list1 = [1, 2, 3]
list2 = ['a', 'b']

FOR each item1 IN list1:
    FOR each item2 IN list2:
        PRINT item1, item2
This code will output: 
1 a
1 b
2 a
2 b
3 a
3 b

Common Uses

  • Processing nested collections (such as lists of lists).
  • Comparing pairs of values.
  • Working with grid or matrix-like data.
